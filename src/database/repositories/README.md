# Database Repositories

Custom TypeORM repositories for complex queries and business logic.

## Repositories
- `base.repository.ts` - Base repository with common methods
- `artist.repository.ts` - Artist-specific queries
- `client.repository.ts` - Client management queries
- `conversation.repository.ts` - Conversation state queries
- `appointment.repository.ts` - Availability and scheduling queries
- `portfolio-image.repository.ts` - Vector similarity search
- `gap-filler-queue.repository.ts` - Queue matching logic
- `analytics.repository.ts` - Business metrics aggregations
