# Notifications Module

Multi-channel notification system.

## Responsibilities
- WhatsApp message sending
- Email notifications
- SMS alerts (optional)
- Notification templates
- Delivery tracking
- Scheduled notifications

## Files
- `notifications.module.ts` - Module definition
- `notifications.service.ts` - Business logic orchestration
- `whatsapp-sender.service.ts` - WhatsApp API integration
- `email-sender.service.ts` - Email service integration
- `template-renderer.service.ts` - Template rendering
- `dto/` - Data transfer objects
