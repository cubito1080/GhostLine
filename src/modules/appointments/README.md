# Appointments Module

Appointment scheduling and calendar management.

## Responsibilities
- Appointment creation and management
- Artist availability checking
- Appointment status tracking (scheduled → confirmed → completed → cancelled)
- Integration with Google Calendar
- Appointment reminders and notifications

## Files
- `appointments.module.ts` - Module definition
- `appointments.service.ts` - Business logic
- `appointments.controller.ts` - REST API endpoints
- `availability.service.ts` - Availability checking logic
- `dto/` - Data transfer objects
