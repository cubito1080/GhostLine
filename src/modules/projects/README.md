# Projects Module

Tattoo project lifecycle management and pricing.

## Responsibilities
- Project creation and management
- Pricing calculations (hourly vs session-based)
- Multi-session project planning
- Deposit management
- Project status tracking (pending → in_progress → completed → cancelled)

## Files
- `projects.module.ts` - Module definition
- `projects.service.ts` - Business logic
- `projects.controller.ts` - REST API endpoints
- `projects.repository.ts` - Data access
- `pricing.service.ts` - Pricing calculation logic
- `dto/` - Data transfer objects
