# Artists Module

Manages artist profiles, configuration, and business settings.

## Responsibilities
- Artist profile management (bio, rates, methodology)
- Style preferences and restrictions
- Calendar configuration (Google Calendar integration)
- Payment settings (Stripe integration)
- Business hours and availability rules

## Files
- `artists.module.ts` - Module definition
- `artists.service.ts` - Business logic
- `artists.controller.ts` - REST API endpoints
- `artists.repository.ts` - Data access
- `dto/` - Data transfer objects
