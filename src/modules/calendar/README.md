# Calendar Module

Google Calendar integration and synchronization.

## Responsibilities
- Google Calendar API integration
- Appointment synchronization
- Availability checking from calendar
- Calendar event creation/updates
- OAuth token management

## Files
- `calendar.module.ts` - Module definition
- `calendar.service.ts` - Business logic
- `google-calendar.service.ts` - Google Calendar API wrapper
- `appointment.repository.ts` - Appointment data access
- `dto/` - Data transfer objects
