# Clients Module

Customer relationship management (CRM) for tattoo clients.

## Responsibilities
- Client profile management
- Contact information and preferences
- Project history tracking
- Client status pipeline (lead → prospect → active → completed)
- Communication history

## Files
- `clients.module.ts` - Module definition
- `clients.service.ts` - Business logic
- `clients.controller.ts` - REST API endpoints
- `clients.repository.ts` - Data access
- `dto/` - Data transfer objects
