# Gemini Function Calling Tools

Tools for Gemini AI function calling capabilities.

## Tools
- `check-availability.tool.ts` - Check artist availability
- `generate-payment-link.tool.ts` - Create Stripe payment links
- `search-portfolio.tool.ts` - Search portfolio images
- `calculate-pricing.tool.ts` - Calculate project pricing
- `schedule-appointment.tool.ts` - Schedule appointments

Each tool follows the Gemini function calling schema.
