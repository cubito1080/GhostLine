# Analytics Module

Business intelligence and event tracking.

## Responsibilities
- Event tracking (prospect interactions, conversions, etc.)
- Business metrics aggregation
- Revenue analytics
- Client lifecycle analytics
- Conversation funnel tracking
- Performance dashboards

## Files
- `analytics.module.ts` - Module definition
- `analytics.service.ts` - Business logic
- `analytics.controller.ts` - REST API endpoints
- `events.service.ts` - Event tracking
- `metrics.service.ts` - Metrics calculation
- `dto/` - Data transfer objects
