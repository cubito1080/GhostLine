# Sessions Module

Individual tattoo session management.

## Responsibilities
- Session tracking for multi-session projects
- Session duration and progress recording
- Session notes and documentation
- Session status tracking (scheduled → in_progress → completed → cancelled)
- Integration with healing checkpoints

## Files
- `sessions.module.ts` - Module definition
- `sessions.service.ts` - Business logic
- `sessions.controller.ts` - REST API endpoints
- `dto/` - Data transfer objects
